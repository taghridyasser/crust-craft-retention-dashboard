# 🥐 Crust & Craft Retention Dashboard

## 📌 Project Overview
This project analyzes customer retention and promotional campaign effectiveness for an artisan bakery (Crust & Craft).  
The goal is to understand customer behavior, improve re-engagement strategies, and reduce churn using data-driven insights.

---

## 🎯 Objectives
- Measure SMS campaign effectiveness (click → return within 7 days)
- Analyze customer return intervals
- Identify high-risk churn segments
- Optimize promotion timing based on foot traffic trends
- Understand product purchase cycles

---

## 📊 Dashboard Features
- 📈 Campaign Performance Tracking (CTR vs Return Rate)
- 🔁 Customer Return Behavior Analysis (7-day window)
- ⚠️ Churn Risk Segmentation (based on visit gaps)
- 📅 Foot Traffic Trends (daily insights)
- 🛍️ Purchase Cycle Analysis

---

## 🛠️ Tools & Technologies
- Tableau (Data Visualization)
- Excel / CSV datasets

## 💡 Key Insights
- Customers who engage with SMS campaigns are more likely to return within 7 days
- Longer visit gaps indicate higher churn probability
- Peak foot traffic occurs on specific days → optimal promotion timing
- Certain products drive repeat purchases more effectively

---

## 🚀 Business Impact
This dashboard helps:
- Improve customer retention strategies
- Optimize marketing campaigns
- Increase revenue through targeted promotions
- Enable data-driven decision making
